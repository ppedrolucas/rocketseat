Teste git e github com Linux. "git init" Inicializa o meu repositório local e já começa a monitorar toda a pasta "git rm --cached nomeDoDocumento" faz com que o git pare de monitorar este arquivo em específico "git add nomeDoDocumento ou . " faz o git adicionar todos o repositório ao Stage Area "git restore nomeDoDocumento" caso tenha feito alguma mudança no arquivo e não gostou, esse comando retoma ao útimo estado de alteração. "git commit -m 'seu commit aqui'" "git pull" Puxa as modificações do repositório remoto para o local "git push" Envia as modificações do repositório local para o remoto
